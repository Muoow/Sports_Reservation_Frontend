# Sports Reservation Frontend

## 运行环境要求
- Node.js (建议 v16 及以上)
- npm (或 yarn/pnpm)

## 快速运行

1. 克隆仓库并进入前端目录：
   ```bash
   git clone <your-repo-url>
   cd Sports-reservation-frontend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 打开浏览器访问控制台提示的地址。
